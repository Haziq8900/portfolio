# Haziq Khan | Portfolio Website

A responsive personal portfolio website built with pure HTML5 and CSS3, showcasing my journey as a Software Engineering student — including my projects, skills, education, and certifications.

🔗 **Live Demo:** https://haziq8900.github.io/portfolio/

---

## 🖥️ Theme — "Dark Editor"

The entire site is designed to look and feel like a code editor or terminal, tying the visual identity directly to the developer experience:

- Dark, code-editor-inspired color palette (GitHub Dark inspired)
- Syntax-highlight accent colors (green, blue, purple, orange, pink, red)
- `JetBrains Mono` / `Fira Code` monospace typography throughout
- Faint background grid mimicking an editor canvas
- Section headings styled as code comments (`// About`)
- Window "traffic light" dots in the header logo
- Project cards styled as file tabs (`// file: project.md`)
- Blinking terminal cursor in the hero section
- Contact form styled as a shell script (`$` prompts, `contact-me.sh`)
