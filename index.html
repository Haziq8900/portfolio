Department Management System

Description

Department Management System is a full-stack web-based application designed to efficiently manage academic department operations. The system provides role-based access for Admin, Teachers, and Students, allowing secure management of users, student records, teacher details, courses, and academic results. It is developed using Java (Servlets/JSP), MySQL, HTML, CSS, and JavaScript, following a structured backend architecture. The application includes database integration with relational tables for managing courses, enrollments, marks, and user information. It is deployed using Apache Tomcat Server and demonstrates concepts of database management, web development, and software engineering practices

Bank Management System

Description

Bank Management System is a Java-based desktop application designed to simulate essential banking operations through an interactive graphical user interface. The system allows users to create accounts, securely log in, deposit and withdraw money, check account balance, view mini statements, and manage banking transactions. It is developed using Java Swing for the GUI and MySQL database integration through JDBC for efficient data storage and retrieval. The project follows Object-Oriented Programming (OOP) principles and demonstrates concepts such as database connectivity, event-driven programming, and user authentication. This application provides a simple, secure, and user-friendly platform for managing basic banking activities.
